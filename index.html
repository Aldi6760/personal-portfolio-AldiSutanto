E-Commerce Platform

King Akrilik is a company profile and e-commerce platform for custom acrylic products, designed with a modern and premium appearance. On the frontend side, the website features a cinematic hero slider, product catalog, detailed product pages with direct WhatsApp ordering buttons, an information ticker, and a responsive design built with Bootstrap 5. On the backend side, it provides an admin dashboard to manage products (CRUD), update website content (text, images, contact), and includes an authentication system for security. The technologies used include Laravel, Blade Template, Bootstrap 5, and WhatsApp integration, making the website effective for both branding and simplifying customer transactions.
